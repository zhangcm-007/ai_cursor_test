# 测试用例导出 XMind 说明

本目录用于存放从 **03-测试用例** 导出的 XMind 相关文件。

## 方式一：脚本自动导出（推荐）

1. 安装依赖（首次使用）：
   ```bash
   pip install -r ../tools/requirements.txt
   ```
   或：`pip install xmind`

2. 在框架根目录或任意位置运行：
   ```bash
   python D:\ai_code\testing-framework\tools\export_to_xmind.py
   ```
   或指定某个用例文件：
   ```bash
   python D:\ai_code\testing-framework\tools\export_to_xmind.py "D:\ai_code\testing-framework\03-测试用例\xxx.md"
   ```

3. 生成文件：
   - `*_xmind_outline.txt`：大纲文本，可用 XMind **文件 → 导入 → 大纲** 导入。
   - `*.xmind`：思维导图文件，用 XMind 直接打开。

## 方式二：导入 FreeMind .mm（XMind 不支持直接导入 .txt 时用）

部分 XMind 版本**不支持导入 .txt 大纲**，导入对话框里只显示 **FreeMind Map 文件(*.mm)**。请先用脚本把大纲转成 .mm 再导入：

1. 运行（将本目录下所有 `*_xmind_outline.txt` 转为同名 .mm）：
   ```bash
   python D:\ai_code\testing-framework\tools\outline_to_freemind.py
   ```
   或指定某个大纲文件：
   ```bash
   python D:\ai_code\testing-framework\tools\outline_to_freemind.py "D:\ai_code\testing-framework\04-导出XMind\三角形判定-测试用例_xmind_outline.txt"
   ```
2. 打开 XMind → **文件** → **导入** → 选择 **FreeMind Map 文件(*.mm)**，选中生成的 `*.mm` 文件即可。

本目录下已有 **三角形判定-测试用例.mm** 时，可直接用 XMind 导入该 .mm 文件。

## 方式三：仅使用大纲文本（若支持 .txt 导入）

若你的 XMind 支持「导入 → 大纲」且能选 .txt：

1. 在 **03-测试用例** 的用例文档末尾有「XMind 用大纲」小节，或使用本目录的 `*_xmind_outline.txt`。
2. 在 XMind 中选择 **文件 → 导入 → 大纲**，选择对应 .txt 文件。

## 层级结构说明

- 根节点：项目/模块名称  
- 子节点：每条测试用例（编号 + 标题）  
- 再下一层：优先级、前置条件、测试步骤、预期结果  

导出后可在 XMind 中继续编辑、加图标、分支样式等。
