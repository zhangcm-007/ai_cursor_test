## 详细测试点

### 一、功能测试点

#### 1.1 合法三角形类型判定

| 测试点ID | 测试点描述 | 说明/边界 |
|----------|------------|-----------|
| TP-01 | 等边三角形：三边相等（如 3,3,3）应判定为等边三角形 | 正常 |
| TP-02 | 等腰三角形：仅两边相等（如 3,3,4）、三种组合均需覆盖 | 正常 |
| TP-03 | 一般三角形：三边不等且满足三角形成立条件（如 3,4,5）应判定为一般三角形 | 正常 |
| TP-04 | 不构成三角形：任两边之和 ≤ 第三边（如 1,2,3 或 1,2,4）应判定为“非三角形”或等价提示 | 异常/边界 |
| TP-05 | 边长 ≤ 0：任一边为 0 或负数（如 0,1,2 或 -1,2,3）应给出错误提示，不判为合法三角形 | 异常 |
| TP-06 | 退化三角形：两边之和 = 第三边（如 1,2,3）应视为不构成三角形 | 边界 |
| TP-09 | 输出与预期一致：等边/等腰/一般/非三角形 四种结果与业务定义一致 | 正常 |

#### 1.2 输入与数据

| 测试点ID | 测试点描述 | 说明/边界 |
|----------|------------|-----------|
| TP-07 | 输入非数字：含字母、符号、空字符串、null，应提示输入错误 | 异常 |
| TP-07 | 输入个数异常：非 3 个数（少输、多输）时的处理与提示 | 异常 |
| TP-08 | 小数边长：如 1.5, 2.5, 3.0 等，判定结果与精度处理正确 | 边界 |
| TP-08 | 极大/极小数值：超大数或极小数时的溢出、精度与判定 | 边界 |
